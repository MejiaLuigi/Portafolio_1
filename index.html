INICIO

Bienvenidos a mi portafolio. Soy Luigi Mejia, esta es mi pagina para hablar un poco sobre mi trayecto y el avance que he tenido a raiz de mi estudio intensivo en CESDE - GLOBANT; diseñada basicamente para mostra mi evolución a raíz del comienzo hasta la que llevo.

Esta pagina esta diseñada principalmente con los principio de A11Y (accesibilidad); la cual se enfoca o se dirige basicamente para aquellas personas que cuentas con discapacidad, visual, auditiva, o motriz. Tambien se tienen en cuenta, principios como el responsive (modelos adaptables), para que se pueda ver desde cualquier dispositivo, ampliando asi la forma de leer la información, otro concepto a tener en cuenta al momento de diseñar esta página fue el concepto de la teoría del color y el contraste, para hacerla mucho mas atractiva y llamativa de forma Visual. En esta no se utiliza casí lenguajes de programación como JS o herramientas freemworks como (react, angular, node JS), es una página creada desde 0 con HTML y CSS puro.
